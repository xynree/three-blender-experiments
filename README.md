# Three.js + Blender Exercises

A collection of Blender modeling exercises paired with Three.js visualizations, designed to build skills in 3D modeling, exporting, and interactive web rendering. Each exercise produces a GLB model that is displayed in a SvelteKit + Three.js environment. The final project combines all objects into a cohesive interactive **home studio / office scene**.

### Basic Exercises

- [x] **Water Bottle**  
  - **Blender Concepts:** Primitive modeling, spin modifier, solidify, smooth shading, basic transforms.  
    *Teaches creating 3D objects from 2D profiles, adding thickness, and smoothing curved surfaces.*  
  - **Three.js Concepts:** Loading GLB models, scene setup, camera positioning, ambient & directional lights, object rotation animation.  
    *Teaches importing models, lighting, and simple animation.*

- [ ] **Desk**  
  - **Blender Concepts:** Cube scaling, duplication, grouping, hierarchical modeling.  
    *Teaches assembling objects from primitives and managing hierarchy for multi-part models.*  
  - **Three.js Concepts:** Grouped objects, scene graph hierarchy, positioning, shadows.  
    *Teaches object grouping, scene organization, and shadow casting.*

- [ ] **Coffee Mug**  
  - **Blender Concepts:** Cylinder primitive, inset & extrude, hollow geometry, handle creation, smooth shading.  
    *Teaches hollowing, extrusion, and detail creation on simple objects.*  
  - **Three.js Concepts:** MeshStandardMaterial, shadows, rotation/tilt animation.  
    *Teaches material assignment, shadows, and subtle animations.*

- [ ] **Snack Bowl**  
  - **Blender Concepts:** Cylinder primitive, subdivision surface, proportional editing, smooth curves.  
    *Teaches smoothing, organic shape creation, and proportional adjustments.*  
  - **Three.js Concepts:** PBR materials (roughness & metalness), scene placement, lighting reflections.  
    *Teaches realistic material properties and lighting effects.*

- [ ] **Desk Dice / Paperweights**  
  - **Blender Concepts:** Cube primitive, bevel edges, Boolean difference for dots, symmetry.  
    *Teaches adding detailed features with Booleans and managing symmetry.*  
  - **Three.js Concepts:** Texture mapping, subtle rotations, shadows.  
    *Teaches small-object placement, rotation, and lighting.*

- [ ] **Pencil / Pen**  
  - **Blender Concepts:** Cylinder primitive, extrusion, multi-material assignment (wood, lead, paint), slight bending.  
    *Teaches multi-material objects and minor deformations for realism.*  
  - **Three.js Concepts:** Multi-material objects, orbit controls, subtle floating or tilting animation.  
    *Teaches material handling and adding interactive motion.*

- [ ] **Office Chair**  
  - **Blender Concepts:** Low-poly modeling, mirror modifier, mesh joining.  
    *Teaches low-poly design and using symmetry for efficiency.*  
  - **Three.js Concepts:** Instancing with `InstancedMesh`, positioning multiple objects, shadows.  
    *Teaches performance-friendly duplication and placement.*

- [ ] **Decorative Torus**  
  - **Blender Concepts:** Torus primitive, subdivision modifier, smooth shading, minor sculpting.  
    *Teaches smoothing and minor sculpting for decorative objects.*  
  - **Three.js Concepts:** Animation loops, rotation, scale pulsation, emissive materials.  
    *Teaches dynamic visual effects and object animation.*

- [ ] **Laptop**  
  - **Blender Concepts:** Multi-part modeling (base + screen), bevel edges, separate materials for base & screen.  
    *Teaches assembling multi-part objects and assigning multiple materials.*  
  - **Three.js Concepts:** Multi-part GLB handling, material tweaks, subtle animations.  
    *Teaches handling multi-material models and subtle interactive animation.*

- [ ] **Small Plant / Desk Decoration**  
  - **Blender Concepts:** Organic modeling, sculpting, decimation, stylized shapes, material assignment.  
    *Teaches sculpting irregular shapes and stylized low-poly modeling.*  
  - **Three.js Concepts:** Material application, placement, optional subtle rotation or floating animation.  
    *Teaches positioning decorative objects and adding minor motion.*

- [ ] **Low-Poly Rock / Decorative Stone**  
  - **Blender Concepts:** Sculpting basic shapes, decimation for low-poly style, flat shading.  
    *Teaches low-poly stylization and sculpting for desk decorations.*  
  - **Three.js Concepts:** Shadows, lighting, subtle rotation/scale animation.  
    *Teaches scene integration for small objects and subtle animations.*

### Final Project

- [ ] **Interactive Home Studio / Office Scene**  
  - **Blender Concepts:** Consolidating all previous modeling skills, scaling, transforms.  
    *Teaches assembling multiple objects into a cohesive scene with consistent scale and transforms.*  
  - **Three.js Concepts:** Scene setup, object grouping, lights & shadows, PBR materials, orbit controls, subtle animations, optional interactivity (hover/click).  
    *Teaches building a full interactive scene, integrating objects, lighting, and animation.*
